## Tecnologias utilizadas
- SQL 🛠️
- MySQL 🛠️
- Lombok
- Log4j2
- Maven
- Java
- Mapstruct 🛠️
- JPA 🛠️
- Hibernate 🛠️
- JPQL 🛠️
- Criteria 🛠️
- Junit5 🛠️
- Mockito 🛠️
- Docker (Container MySQL) 🛠️
